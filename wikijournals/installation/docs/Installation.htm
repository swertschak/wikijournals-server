(for version 3.0.0 or above)

A new installation of ApPHP EasyInstaller (PHP EI) is a very straight forward process:

[top]

Step 1. Uncompressing downloaded archive.

Uncompress the PHP EI version 3.x.x script archive.
The archive will create a directory called "PHPEI_2xx".


[top]

Step 2. Uploading files.

Copy file install.php and folder, named install/ (including all its content) into directory of your script. For example, if your script (program) is in directory, named MyProgram - copy install.php and folder install/ into MyProgram/ directory.


That is all! PHP EasyInstaller v3.x.x. was installed and now ready to work! Lets see now Getting Started!
